## 1. 概述

将英语语法学习进度以3D球形技能树的形式展示，每个语法知识点作为一个可点亮的技能点，用户通过拖拽旋转球体来查看学习进度。

### 目标
- 将抽象的学习进度转化为直观的"技能树"视觉体验
- 激励用户通过点亮更多技能点来保持学习动力
- 提供新奇有趣的交互体验

## 2. 数据模型

### 2.1 现有数据结构

```kotlin
// GrammarLessonEntity - 技能点
@Entity(tableName = "grammar_lessons")
data class GrammarLessonEntity(
    @PrimaryKey val id: Long,
    val stageId: Long,         // 所属阶段
    val title: String,         // 技能点名称
    val order: Int,           // 阶段内顺序
    val isCompleted: Boolean = false,
    val correctRate: Float = 0f,    // 正确率
    val lastPracticedAt: Long? = null
)
```

### 2.2 技能点扩展

新增熟练度字段：
- `proficiency: Int (0-100)` - 熟练度百分比
- `learnedAt: Long?` - 首次学习时间
- `connections: List<Long>` - 关联的其他技能点ID

### 2.3 技能点关系

| 关系类型 | 说明 |
|---------|------|
| 阶段内顺序 | 同一阶段内的lesson按order顺序连接 |
| 跨阶段依赖 | 后续阶段的关键知识点可关联前置阶段 |
| 相关知识点 | 基于语法内容的相关性（如：名词→代词→冠词） |

## 4. 3D球形可视化

### 4.1 节点分布算法

使用Fibonacci Sphere算法将N个技能点均匀分布在球面上：

```kotlin
// 对第i个节点（从0开始）
phi = acos(1 - 2 * (i + 0.5) / n)  // 垂直角度
theta = PI * (1 + sqrt(5.0)) * i   // 水平角度
```

### 4.2 3D投影到2D

使用透视投影将3D坐标转换为2D屏幕坐标：

```kotlin
// 透视投影公式
scale = fov / (fov + z)
x2d = x * scale + centerX
y2d = y * scale + centerY
```

### 4.3 节点渲染

| 属性 | 计算方式 |
|-----|---------|
| 位置 | 球面坐标(theta, phi)转3D笛卡尔坐标 |
| 大小 | 基础大小 * scale (近大远小) |
| 亮度 | 基础亮度 * proficiency/100 |
| 层级 | 根据z值确定绘制顺序和zIndex |
| 可见性 | z > 0 (背面节点不可见) |

### 4.4 连接线渲染

- 只渲染前向连接（从已学习到未学习）
- 连接线颜色与源节点熟练度相关
- 使用Canvas绘制贝塞尔曲线

## 5. 交互设计

### 5.1 旋转控制

| 操作 | 行为 |
|-----|------|
| 水平拖拽 | 绕Y轴旋转球体 |
| 垂直拖拽 | 绕X轴旋转球体 |
| 松开手指 | 保留旋转惯性，逐渐减速 |

### 5.2 自动旋转

- 默认开启缓慢自动旋转
- 用户操作时暂停自动旋转
- 3秒无操作后恢复自动旋转

### 5.3 节点交互

| 操作 | 行为 |
|-----|------|
| 点击节点 | 跳转到对应lesson学习页面 |
| 长按节点 | 显示技能点详情tooltip |

### 5.4 视角控制

- 支持双指缩放调整球体大小
- 双击重置到默认视角

## 7. 技术实现

### 7.1 实现方案

使用纯Compose Canvas实现，无需额外3D库：

1. **球面坐标计算** - Fibonacci Sphere算法
2. **3D旋转** - 旋转矩阵变换
3. **透视投影** - 3D到2D坐标转换
4. **Canvas绘制** - 节点和连接线渲染
5. **手势处理** - DragGesture检测旋转

### 7.2 依赖引入

无额外依赖，使用Compose原生API：
- `Canvas` - 3D投影渲染
- `GestureDetector` - 手势处理
- `remember` / `mutableStateOf` - 状态管理
- `animateFloatAsState` - 动画过渡
