## 详细笔记

### 一、音标概述

#### 什么是音标？
- 音标是标注发音的符号，就像汉语的拼音
- 英语音标分为**辅音**和**元音**两大类
- 掌握音标是准确发音的基础

#### 48个国际音标
- 20个元音（12个单元音 + 8个双元音）
- 28个辅音

### 五、连读与失去爆破

#### 1. 连读
- 前一个词以辅音结尾，后一个词以元音开头 → 连读
- 例：not at all → /nɒtətɔːl/

#### 2. 失去爆破
- 爆破音（/p/, /b/, /t/, /d/, /k/, /g/）后接其他辅音时，只做口型不发音
- 例：good boy → /ɡʊd bɔɪ/ （d 不完全爆破）

### 六、常见发音难点

#### 1. /θ/ vs /s/
- /θ/：think, thing, thank
- /s/：sink, sing, sank

#### 2. /ð/ vs /z/
- /ð/：this, that, the
- /z/：zoo, rise, these

#### 3. /l/ vs /n/
- /l/：light, night, like
- /n/：nine, knife, night

#### 4. 元音 /æ/ vs /e/
- /æ/：bad, sad, man（嘴巴张大）
- /bed/：bed, red, ten（嘴巴半开）

### 例句

#### 单元音练习
1. /ɪ/ — sit, fit, bit, hit
2. /e/ — bed, leg, ten, men
3. /æ/ — cat, bag, man, map
4. /ə/ — about, teacher, China
5. /ʌ/ — cup, bus, love, money

#### 双元音练习
1. /eɪ/ — make, name, cake, same
2. /aɪ/ — time, find, five, bike
3. /əʊ/ — home, go, no, phone
4. /aʊ/ — house, out, down, now

#### 辅音练习
1. /θ/ — think, thank, thin, thing
2. /ð/ — this, that, the, there
3. /ʃ/ — ship, she, wash, dish
4. /tʃ/ — child, choose, watch, catch

#### 对比练习
- /θ/ vs /s/: think / sink / thank / sank
- /ð/ vs /z/: this / zoo / they / zoo
- /l/ vs /n/: light / night / like / bike

### 注意事项

1. **元音是发音的核心**，英语中元音发音准确与否直接影响听力理解和口语交流。

2. **双元音要饱满**，从第一个元音滑向第二个元音：
   - /eɪ/ → 从 /e/ 滑向 /ɪ/
   - /aɪ/ → 从 /a/ 滑向 /ɪ/

3. **清辅音和浊辅音的区别在于声带振动**：
   - 清辅音：声带不振动（/p/, /t/, /k/, /f/, /s/）
   - 浊辅音：声带振动（/b/, /d/, /g/, /v/, /z/）

4. **/θ/ 和 /ð/ 是英语特有的音**，很多学习者会用 /s/ 和 /z/ 替代，注意舌尖要放在两齿之间。

5. **字母在单词中的发音不是固定的**，需要根据位置和规则记忆：
   - A: name (/eɪ/) vs cat (/æ/)
   - O: home (/əʊ/) vs dog (/ɒ/)

6. **连读和失去爆破是流利英语的关键**，多听原版材料并模仿。
