# 主语从句

## 基本信息

| 属性 | 内容 |
|------|------|
| 序号 | 30 |
| 语法点 | 主语从句 |
| 核心内容 | That/Whether/疑问词引导的从句作主语 |

## 常用词组/搭配

It is + adj. + that..., What matters is...

## 详细笔记

### 一、主语从句定义

- **定义**：在句子中作主语的从句称为主语从句
- **位置**：位于句首或用 it 作形式主语

### 二、主语从句的引导词

#### 1. 从属连词 that / whether
- **That** he is honest is true. （that 无意义，可省略）
- **Whether** he will come is unknown. （whether 表是否）

#### 2. 疑问代词
- **Who** will be the winner is not clear.
- **What** we need is more time.
- **Which** team will win is hard to say.
- **Whose** book this is remains unknown.

#### 3. 疑问副词
- **When** we will leave is not decided.
- **Where** we should go is a question.
- **Why** he was late is not clear.
- **How** we can solve this problem is important.

### 三、主语从句的位置

#### 1. 置于句首
- **That** he finished the work on time made us happy.
- **What** I need is your support.

#### 2. 用 it 作形式主语
- **It** is true **that** he is honest.
- **It** is important **that** we should study hard.
- **It** is a pity **that** you can't come.
- **It** is said **that** he is a great scientist.

#### 3. 常见结构

| 结构 | 用法 |
|------|------|
| It is + 形容词 + that | It is necessary that... |
| It is + 名词 + that | It is a pity that... |
| It is + 过去分词 + that | It is said that... |
| It + 动词 + that | It seems that... |
| It + 不及物动词 + that | It happened that... |

### 四、what 引导的主语从句

#### 1. what = the thing(s) which
- **What** you said is right. （你说的是对的）
- **What** I want is a cup of coffee. （我想要的是一杯咖啡）

#### 2. what 引导的主语从句，主谓一致
- **What** he needs **is** time. （单数）
- **What** we need **are** money and support. （复数）

### 五、主语从句的时态

- 主语从句的时态受主句影响，但不严格限制
- **That** he **came** late surprised everyone. （过去）
- **That** he **will come** tomorrow is sure. （将来）

### 六、it 作形式主语的常见句型

| 句型 | 例句 |
|------|------|
| It is important that... | It is important that we work together. |
| It is necessary that... | It is necessary that everyone should attend. |
| It is possible that... | It is possible that he will come. |
| It is strange that... | It is strange that she is not here. |
| It is reported that... | It is reported that the economy is growing. |
| It seems that... | It seems that he is tired. |
| It happened that... | It happened that I was there. |
| It doesn't matter that... | It doesn't matter that you are late. |

### 例句

#### that 引导
1. **That** he is a good teacher is true.
2. **That** she will come is certain.
3. It is important **that** we study hard.

#### whether 引导
1. **Whether** he will come is unknown.
2. It is not clear **whether** she is right.

#### 疑问代词引导
1. **What** you need is more practice.
2. **Who** will win the game is uncertain.
3. **Which** one to choose is difficult.

#### 疑问副词引导
1. **When** we will have the meeting is not decided.
2. **Where** we should go is still a question.
3. **Why** he did this is a mystery.

#### it 作形式主语
1. **It** is important **that** we work together.
2. **It** is a pity **that** you can't come.
3. **It** is said **that** he is a great scientist.
4. **It** seems **that** something is wrong.
5. **It** happened **that** I was at home that day.

#### what 引导的主语从句
1. **What** she said is true.
2. **What** I want is a cup of water.
3. **What** we need are more resources.

### 注意事项

1. **主语从句的引导词 that 不能省略**（位于句首时）：
   - **That** he is honest is true. （不能省略）
   - It is true **(that)** he is honest. （可省略）

2. **主语从句的语序**：用陈述语序，不用疑问语序。
   - 错误：**What is he** is unknown.
   - 正确：**What he is** is unknown.

3. **whether vs. if**：
   - 主语从句中用 whether，不用 if
   - 正确：**Whether** he will come is unknown.
   - 错误：**If** he will come is unknown.

4. **主语从句后的谓语动词**：用单数第三人称。
   - **What** he says **is** true.
   - **What** we need **are** books. （复数概念）

5. **it 作形式主语时 that 从句中的虚拟语气**：
   - It is important that everyone **(should)** attend the meeting.
   - 有些形容词后的 that 从句用 should（虚拟语气）。

### 练习题

#### 一、用适当的引导词填空

1. ____ he is honest is true.
2. ____ we will go is not decided.
3. ____ you need is more practice.
4. ____ he will come is unknown.
5. ____ we should do is important.

#### 二、选择题

1. ____ is true that he is a good student.
   A. What B. That C. Which D. Who

2. It is important ____ we study hard.
   A. that B. what C. which D. whether

3. ____ we will have a meeting is not decided yet.
   A. What B. That C. Whether D. Which

4. It is said ____ he is a great scientist.
   A. that B. what C. which D. who

5. ____ is unclear who will win the competition.
   A. That B. This C. It D. What

#### 三、按要求改写句子

1. That he is right is true. （用 it 改写）
   _________________________________________________
2. He will come tomorrow. It seems that... （合并）
   _________________________________________________
3. We need more time. What we need is... （合并）
   _________________________________________________
4. Whether we will go is uncertain. （用 it 改写）
   _________________________________________________

#### 四、答案

**一、**
1. That    2. Whether/When    3. What    4. Whether    5. What

**二、**
1. B（that 引导主语从句）
2. A（it is important that...）
3. C（whether 引导主语从句）
4. A（it is said that...）
5. C（it 作形式主语）

**三、**
1. It is true that he is right.
2. It seems that he will come tomorrow.
3. What we need is more time.
4. It is uncertain whether we will go.
