# 被动语态

## 基本信息

| 属性 | 内容 |
|------|------|
| 序号 | 23 |
| 语法点 | 被动语态 |
| 核心内容 | 各种时态的被动形式 |

## 常用词组/搭配

be made of/from, be known as, be supposed to

## 详细笔记

### 一、被动语态的定义与构成

#### 1. 定义
- 当句子的主语是动作的承受者时，使用被动语态
- 主动语态：主语 + 动词 + 宾语
- 被动语态：主语 + be + 过去分词 + (by + 动作执行者)

#### 2. 构成
- **结构**：主语 + be + 过去分词
- **例**：The book **is written** by him. / The work **was done** by us.

### 二、各时态的被动语态

| 时态 | 主动结构 | 被动结构 |
|------|----------|----------|
| 一般现在时 | write | am/is/are written |
| 一般过去时 | wrote | was/were written |
| 一般将来时 | will write | will be written |
| 现在进行时 | is writing | is being written |
| 过去进行时 | was writing | was being written |
| 现在完成时 | have written | have been written |
| 过去完成时 | had written | had been written |
| 将来完成时 | will have written | will have been written |

### 三、被动语态的各种句型

#### 1. 肯定句
- The letter **was sent** yesterday.
- The work **is being done** now.

#### 2. 否定句
- The letter **was not sent** yesterday.
- The work **is not being done** now.

#### 3. 一般疑问句
- **Was** the letter **sent** yesterday?
- **Is** the work **being done** now?

#### 4. 特殊疑问句
- **When was** the letter **sent**?
- **Who was** the letter **sent by**?

### 四、双宾语的被动语态

#### 1. 主动语态中的双宾语
- give sb. sth. / give sth. to sb.
- send sb. sth. / send sth. to sb.

#### 2. 两种被动形式
- He gave me a book. → I **was given** a book. / A book **was given to** me.
- She told him a story. → He **was told** a story. / A story **was told to** him.

### 五、情态动词的被动语态

- **结构**：情态动词 + be + 过去分词
- The work **must be done** today.
- The book **can be finished** tomorrow.
- The window **should be fixed** now.
- The problem **may be solved** soon.

### 六、被动语态与介词搭配

| 搭配 | 含义 | 例句 |
|------|------|------|
| be made of | 看得见原材料 | The desk **is made of** wood. |
| be made from | 看不见原材料 | Paper **is made from** wood. |
| be known as | 作为……闻名 | He **is known as** a writer. |
| be used for | 用于 | This tool **is used for** cutting. |
| be used to | 习惯于 | She **is used to** getting up early. |
| be supposed to | 应该 | You **are supposed to** finish it. |
| be interested in | 对……感兴趣 | I **am interested in** reading. |
| be satisfied with | 对……满意 | They **are satisfied with** the result. |

### 七、主动形式表示被动意义

| 结构 | 用法 | 例句 |
|------|------|------|
| want/need + doing | 需要被…… | The book **needs reading** again. |
| be worth + doing | 值得…… | The movie **is worth watching**. |
| be easy/difficult/hard to do | 易于/难于…… | The question **is easy to answer**. |
| feel/look/smell/taste + adj. | 感官动词 | The cloth **feels soft**. |

### 例句

#### 一般时态的被动
1. The book **is written** by him. （一般现在时）
2. The letter **was sent** yesterday. （一般过去时）
3. The work **will be done** tomorrow. （一般将来时）

#### 进行时的被动
1. The house **is being built** now. （现在进行时）
2. The letter **was being written** when I came in. （过去进行时）

#### 完成时的被动
1. The work **has been finished**. （现在完成时）
2. The work **had been finished** before I came. （过去完成时）

#### 情态动词的被动
1. The problem **must be solved** immediately.
2. The book **can be read** by everyone.
3. The window **should be fixed** soon.
4. The work **may be done** tomorrow.

#### 双宾语的被动
1. He gave me a book. → I **was given** a book. / A book **was given to** me.
2. She told them a secret. → They **were told** a secret. / A secret **was told to** them.

#### 介词搭配
1. The table **is made of** wood. （看得见原材料）
2. Wine **is made from** grapes. （看不见原材料）
3. He **is known as** a great scientist.
4. This phone **is used for** calling.

### 注意事项

1. **被动语态中 be 动词不可省略**：
   - 错误：Book read by me.
   - 正确：Book is read by me.

2. **不及物动词没有被动语态**：
   - 错误：The accident was happened last night.
   - 正确：The accident happened last night.

3. **主动形式表被动意义的用法**：
   - want/need + doing = want/need to be done
   - The room needs cleaning. = The room needs to be cleaned.

4. **be made of vs. be made from**：
   - be made of：物理变化，看得见原材料
   - be made from：化学变化，看不见原材料

5. **by + 动作执行者可省略**：当不强调动作执行者时，by 短语可省略。

6. **双宾语变被动时，优先变间接宾语**：He gave me a book. → I was given a book. （更常见）

#### 三、用所给词的适当形式填空

1. The book ____ (read) by everyone.
2. The work must ____ (do) tomorrow.
3. The house ____ (build) last year.
4. The problem ____ (solve) already.
5. The window is being ____ (fix) now.

#### 四、答案

**一、**
1. The house is built every year.
2. A letter was written by her yesterday.
3. The work will be finished tomorrow.
4. The book is being read by him now.
5. The project has been finished (by them).

**二、**
1. D（一般过去时的被动）
2. A（现在进行时的被动）
3. D（一般过去时的被动）
4. A（be known as 作为……闻名）
5. D（be made of 由……制成）

**三、**
1. is read
2. be done
3. was built
4. has been solved
5. fixed
