# 目的与结果状语从句

## 基本信息

| 属性 | 内容 |
|------|------|
| 序号 | 22 |
| 语法点 | 目的与结果状语从句 |
| 核心内容 | so that, such...that, so...that |

## 常用词组/搭配

in order that, in order to

## 详细笔记

### 一、目的状语从句

#### 1. 定义与结构

- **定义**：说明主句动作目的的从句
- **结构**：主句 + so that/in order that + 从句

#### 2. 目的状语连词

| 连词 | 用法 | 例句 |
|------|------|------|
| so that | 引导目的从句 | I studied hard **so that** I could pass the exam. |
| in order that | 正式 | I worked overtime **in order that** I could finish the work. |
| that | 较口语 | He spoke slowly **that** everyone could understand. |

#### 3. 目的状语中的 can/could/may/might

- 从句中常含 can/could/may/might
- 例：I left early **so that** I **could** catch the train.

#### 4. 动词不定式表目的

- **结构**：to do / in order to / so as to
- **例**：I studied hard **to pass** the exam. = I studied hard **in order to** pass the exam.

### 二、结果状语从句

#### 1. 定义与结构

- **定义**：说明主句动作结果的从句
- **结构**：主句 + so that + 从句 / such...that... / so...that...

#### 2. 结果状语连词

| 结构 | 用法 | 例句 |
|------|------|------|
| so that | 表结果 | It was dark, **so that** I couldn't see anything. |
| so...that | 如此……以至于 | He is **so tall** **that** he can reach the ceiling. |
| such...that | 如此……以至于 | It is **such an interesting** book **that** I read it twice. |

#### 3. so vs. such

| 类型 | 结构 | 例句 |
|------|------|------|
| so + 形容词/副词 | so + adj./adv. + that | so beautiful that... |
| so + many/much/little + 名词 | so + many/few + 名词 | so many books that... |
| such + a/an + 形容词 + 名词 | such + (a/an) + adj. + n. | such a beautiful girl that... |
| such + 复数/不可数名词 | such + n. (pl./uncount) | such good students that... |

### 三、目的 vs. 结果的区别

| 类型 | 含义 | 位置 |
|------|------|------|
| so that + can/could/may/might | 表目的（可能） | 从句中含情态动词 |
| so that + 陈述语气 | 表结果 | 从句中无情态动词 |

- **目的**：I got up early **so that** I **could** catch the train. （为了能赶上火车）
- **结果**：I got up early, **so that** I **caught** the train. （结果赶上了）

### 四、常用句型

#### 1. too...to（太……而不能……）
- He is **too tired to walk**. = He is so tired that he cannot walk.

#### 2. enough to（足够……可以……）
- He is **old enough to go** to school. = He is so old that he can go to school.

#### 3. so...as to（如此……以至于……）
- Would you be **so kind as to** help me? （请你帮个忙好吗？）

### 五、注意事项

#### 1. 目的和结果的区分
- 主句 + so that + 从句（可能表目的或结果，需根据上下文判断）
- 从句中若有 can/could/may/might，通常表目的
- 从句中若无情态动词，通常表结果

#### 2. so that 可引导目的和结果从句
- 目的：I locked the door **so that** no one **could** enter.
- 结果：I locked the door, **so that** everyone **was** locked out.

### 例句

#### 目的状语从句
1. I studied hard **so that** I **could** pass the exam.
2. He got up early **in order that** he **might** catch the train.
3. She saved money **so that** she **could** buy a car.
4. I called you **so that** you **wouldn't** worry.

#### 动词不定式表目的
1. I **to pass** the exam, I studied hard.
2. **In order to** catch the train, he got up early.
3. **So as to** not be late, I left early.

#### 结果状语从句
1. It was dark, **so that** I couldn't see anything.
2. He is **so tall** **that** he can reach the ceiling.
3. She is **such a beautiful** girl **that** everyone likes her.
4. It was **such interesting** news **that** everyone was excited.

#### so vs. such
1. It is **so beautiful** a flower. / It is **such a beautiful** flower.
2. There are **so many** people. / There are **such many** people. （错误）
3. It is **so interesting** a book. / It is **such an interesting** book.

#### too...to / enough to
1. He is **too tired to walk**. = He is so tired that he cannot walk.
2. She is **old enough to go** to school. = She is so old that she can go to school.
3. The book is **too difficult to understand**.

### 注意事项

1. **so that 既可表目的也可表结果**：
   - 表目的：I left early **so that** I **could** catch the train.（从句有 could）
   - 表结果：I left early, **so that** I **caught** the train.（从句无情态动词）

2. **so 和 such 的区别**：
   - so + 形容词/副词：so beautiful, so fast
   - so + many/much/little + 名词：so many books, so much money
   - such + a/an + 形容词 + 名词：such a beautiful flower
   - such + 形容词 + 复数/不可数名词：such good students, such water

3. **such 修饰不可数名词时不能加 a/an**：
   - 错误：such a water
   - 正确：such water

4. **too...to 和 enough to 的否定**：
   - He is too old to work. = He is not old enough to work.
   - 两者都表示"太……而不能……"

5. **in order to 和 so as to 的区别**：
   - in order to 可放句首或句中
   - so as to 通常不放句首
   - In order to succeed, you must work hard. ✓
   - So as to succeed, you must work hard. ✗

### 练习题

#### 一、用 so that, such...that, so...that 填空

1. He is ____ tall ____ he can reach the ceiling.
2. It is ____ interesting a book ____ I read it twice.
3. It rained hard, ____ we stayed at home.
4. She is ____ beautiful ____ everyone likes her.
5. He worked hard ____ he could pass the exam.

#### 二、选择题

1. It is ____ interesting movie that everyone wants to see it.
   A. so B. such C. such an D. so an

2. He is ____ old to go to school.
   A. too B. very C. quite D. so

3. I got up early ____ I could catch the train.
   A. so that B. such that C. so D. such

4. She is ____ a clever girl that she can solve this problem.
   A. so B. such C. too D. very

5. He was ____ tired ____ he fell asleep.
   A. so, that B. such, that C. too, to D. enough, to

#### 三、按要求改写句子

1. He is very tall. He can reach the ceiling. （用 so...that 合并）
   _________________________________________________
2. It was a very interesting book. I read it twice. （用 such...that 改写）
   _________________________________________________
3. I studied hard. I wanted to pass the exam. （用 so that 合并）
   _________________________________________________
4. He is too tired to walk. （用 enough 改写）
   _________________________________________________

#### 四、答案

**一、**
1. so, that    2. such an, that    3. so that    4. such, that    5. so that

**二、**
1. C（such an + 形容词 + 单数名词）
2. A（too...to 句型）
3. A（so that + can/could 表目的）
4. B（such a + 形容词 + 单数名词）
5. A（so...that 表结果）

**三、**
1. He is so tall that he can reach the ceiling.
2. It is such an interesting book that I read it twice.
3. I studied hard so that I could pass the exam.
4. He is not old enough to walk. / He is tired enough to fall asleep.
