# 条件状语从句

## 基本信息

| 属性 | 内容 |
|------|------|
| 序号 | 21 |
| 语法点 | 条件状语从句 |
| 核心内容 | if/unless |

## 常用词组/搭配

as long as, provided that, in case

## 详细笔记

### 一、条件状语从句定义

- **定义**：用从属连词引导，表示某种条件下的结果
- **结构**：主句 + 从属连词 + 从句 / 从属连词 + 从句 + 主句

### 二、常见条件状语连词

#### 1. if（如果）

- **用法**：最常用的条件连词
- **注意**：主句用将来时，从句用一般现在时代替将来时
- **例**：**If** it rains tomorrow, I will stay at home.

#### 2. unless（除非、如果不）

- **用法**：相当于 if not
- **例**：I won't go **unless** you come with me. = I won't go **if** you don't come with me.

#### 3. as long as / so long as（只要）

- **例**：You can use my bike **as long as** you return it on time.

#### 4. provided that / providing that（如果、只要）

- **例**：**Provided that** you work hard, you will succeed.

#### 5. in case（以防、万一）

- **例**：Take an umbrella **in case** it rains.

#### 6. on condition that（在……条件下）

- **例**：I will help you **on condition that** you keep it secret.

#### 7. suppose / supposing（假设）

- **例**：**Supposing** it rains, what will you do?

### 三、真实条件句 vs. 虚拟条件句

#### 1. 真实条件句（真实可能）

- **结构**：if 从句用现在时，主句用将来时
- **例**：If you study hard, you will pass the exam.

#### 2. 虚拟条件句（非真实、假设）

- **结构**：if 从句用过去时，主句用 would/should/could/might + 动词原形
- **例**：If I were you, I would go. （与现在事实相反）

### 四、if 条件句的时态搭配

| 从句 | 主句 | 例句 |
|------|------|------|
| 一般现在时 | 一般将来时 | If it rains, I will stay home. |
| 现在完成时 | 将来完成时 | If you have finished, you can leave. |
| 过去时 | would + 动词 | If I had money, I would buy it. |

### 五、if 的省略用法

#### 1. if 省略 if
- If you need help, call me. → Need help? Call me.
- If you are hungry, eat this. → Hungry? Eat this.

#### 2. if any / if ever
- If any, call me. （如果有的话，打电话给我）

### 六、条件句的倒装

#### 1. 省略 if，用倒装
- If I were you, I would go. → **Were** I you, I would go.
- If you had worked harder, you would have passed. → **Had** you worked harder, you would have passed.

#### 2. should 放句首
- If it should rain, I will stay home. → **Should** it rain, I will stay home.

### 例句

#### if
1. **If** it rains tomorrow, I will stay at home.
2. **If** you study hard, you will pass the exam.
3. If you **need** help, call me.
4. If she **comes**, let me know.

#### unless
1. I won't go **unless** you come with me.
2. You can't pass the exam **unless** you study hard.
3. I won't believe it **unless** I see it with my own eyes.

#### as long as / so long as
1. You can borrow my book **as long as** you return it on time.
2. **So long as** you are happy, I am happy.
3. I will help you **as long as** you try your best.

#### provided that
1. **Provided that** you follow the rules, you can join us.
2. **Providing that** it doesn't rain, we will go out.

#### in case
1. Take an umbrella **in case** it rains.
2. Bring your ID **in case** you need it.

#### 条件句时态
1. If he **comes**, I will tell him. （现在时 → 将来时）
2. If you **have finished**, you can leave. （现在完成时）
3. If I **were** you, I **would** go. （虚拟语气）
4. If she **had studied**, she **would have passed**. （过去虚拟）

### 注意事项

1. **主句用将来时，从句用一般现在时代替将来时**：
   - 错误：If it will rain tomorrow, I will stay home.
   - 正确：If it rains tomorrow, I will stay home.

2. **unless = if not**：
   - 错误：Unless you don't work hard, you will fail.
   - 正确：Unless you work hard, you will fail. / If you don't work hard, you will fail.

3. **as long as vs. if**：
   - as long as 强调条件，if 强调假设
   - As long as you are happy, I am happy.（只要……就……）

4. **真实条件句 vs. 虚拟条件句**：
   - 真实：If I have money, I will buy it.
   - 虚拟：If I had money, I would buy it.

5. **if 条件句中 be 动词的虚拟形式**：
   - 无论主语是什么，be 动词用 were
   - 错误：If I was you...
   - 正确：If I were you...（虚拟语气）

6. **in case 的用法**：in case 后从句用现在时/过去时，不表将来。

### 练习题

#### 一、用适当的连词填空

1. ____ it doesn't rain, we will go out.
2. I will go ____ you come with me.
3. You can pass the exam ____ you study hard.
4. Take an umbrella ____ it rains.
5. ____ you are happy, I am happy.
6. I won't believe it ____ I see it myself.
7. He will help you ____ you ask politely.
8. ____ it should rain, we will cancel the trip.

#### 二、选择题

1. If it ____ tomorrow, I will stay at home.
   A. rains B. will rain C. rain D. raining

2. ____ you work hard, you will succeed.
   A. Unless B. If C. As long as D. Since

3. I won't go to the party ____ you come.
   A. if B. unless C. as long as D. provided

4. You can use my computer ____ you return it on time.
   A. as long as B. unless C. if D. in case

5. Take some money with you ____ you need it.
   A. if B. unless C. as long as D. in case

#### 三、按要求改写句子

1. If you study hard, you will pass the exam. （用 unless 改写）
   _________________________________________________
2. You can come in. You have an invitation. （用 on condition that 合并）
   _________________________________________________
3. I will go. He comes with me. （用 as long as 改写）
   _________________________________________________

#### 四、答案

**一、**
1. If / Unless    2. unless / if    3. if / as long as    4. in case / if    5. As long as / So long as    6. unless / until    7. if / as long as    8. Suppose / Supposing / If

**二、**
1. A（条件从句用一般现在时）
2. C（as long as 表只要）
3. B（unless = if not）
4. A（as long as 表只要）
5. D（in case 以防）

**三、**
1. You won't pass the exam unless you study hard.
2. You can come in on condition that you have an invitation.
3. As long as he comes with me, I will go. / I will go as long as he comes with me.
