## 详细笔记

### 一、四种形式概述

英语动词有四种基本形式，是构成各种时态和语态的基础：

| 形式 | 说明 | 例词 |
|------|------|------|
| **原形** | 动词的基本形式 | work, play, go |
| **第三人称单数** | 一般现在时，主语是第三人称单数 | works, plays, goes |
| **过去式** | 一般过去时 | worked, played, went |
| **过去分词** | 用于完成时和被动语态 | worked, played, gone |
| **现在分词** | 用于进行时 | working, playing, going |

### 二、动词原形（Base Form）

- **用法**：用于一般现在时第一人称、第二人称、复数主语
- **例**：I **work** every day. / You **play** well. / They **go** to school.

### 三、第三人称单数（Third Person Singular）

#### 变化规则

| 规则 | 例词 |
|------|------|
| 一般加 -s | works, reads, eats |
| 以 s, x, z, ch, sh 结尾加 -es | watches, teaches, finishes |
| 以辅音字母 + y 结尾，变 y 为 i + -es | studies, carries, flies |
| 以元音字母 + y 结尾，直接加 -s | plays, stays, buys |
| 以 o 结尾，加 -es | goes, does, watches |

### 四、过去式（Past Simple）

#### 规则变化

| 规则 | 例词 |
|------|------|
| 一般加 -ed | worked, played |
| 以 e 结尾只加 -d | liked, lived |
| 辅音字母 + y 结尾，变 y 为 i + -ed | studied, carried |
| 重读闭音节结尾，双写 + -ed | stopped, planned |

#### 不规则变化类型

| 类型 | 例词 |
|------|------|
| A-A-A型（原型、过去式、过去分词相同） | cut, put, let, set, hurt |
| A-B-B型 | buy → bought → bought |
| A-B-A型 | come → came → come |
| 内部元音变化 | sing → sang → sung |

### 五、过去分词（Past Participle）

#### 规则变化（同过去式）

| 规则 | 例词 |
|------|------|
| 一般加 -ed | worked → worked |
| 以 e 结尾只加 -d | lived → lived |
| 辅音字母 + y 结尾，变 y 为 i + -ed | studied → studied |
| 重读闭音节结尾，双写 + -ed | stopped → stopped |

#### 常见不规则动词表

| 原形 | 过去式 | 过去分词 |

### 六、现在分词（Present Participle）

#### 变化规则

| 规则 | 例词 |
|------|------|
| 一般加 -ing | working, playing |
| 以不发音 e 结尾，去 e 加 -ing | make → making, write → writing |
| 重读闭音节结尾，双写 + -ing | run → running, swim → swimming |
| 以 ie 结尾，变 ie 为 y + -ing | die → dying, tie → tying |

### 七、四种形式在时态中的应用

| 时态 | 构成 |
|------|------|
| 一般现在时 | 原形/三单 |
| 一般过去时 | 过去式 |
| 现在进行时 | be + 现在分词 |
| 过去进行时 | was/were + 现在分词 |
| 现在完成时 | have/has + 过去分词 |
| 过去完成时 | had + 过去分词 |
| 被动语态 | be + 过去分词 |

### 例句

#### 第三人称单数
1. He **works** in a hospital.
2. She **watches** TV every evening.
3. My mother **cooks** delicious food.
4. The cat **eats** fish.

#### 过去式
1. I **worked** late yesterday.
2. She **watched** a movie last night.
3. They **played** football after school.
4. He **went** to Beijing last week.

#### 过去分词（用于现在完成时）
1. I **have finished** my homework.
2. She **has gone** to Beijing.
3. They **have done** the work.
4. He **has written** a letter.

#### 过去分词（用于被动语态）
1. The book **was written** by him.
2. The work **is done** by us.
3. The window **was broken** by the boy.
4. The food **was eaten** by the dog.

#### 现在分词（用于进行时）
1. I **am reading** a book.
2. She **is cooking** dinner.
3. They **are playing** games.
4. He **was sleeping** when I came in.

#### 不规则动词变化
1. go → went → gone: He **has gone** to Shanghai.
2. see → saw → seen: I **have seen** this movie.
3. write → wrote → written: She **wrote** a letter yesterday.
4. eat → ate → eaten: They **have eaten** breakfast.

### 注意事项

1. **第三人称单数只用于一般现在时**，且主语必须是第三人称单数（he/she/it 或单数名词）。

2. **不规则动词需要单独记忆**：常见的 100 多个不规则动词是英语学习的基础。

3. **动词加 -s 和加 -ed 的区别**：
   - -s/-es：第三人称单数（现在时）
   - -ed：过去式/过去分词（过去时）

4. **现在分词的双写规则**：必须是重读闭音节结尾，且末尾只有一个辅音字母。
   - run → running ✓（重读闭音节）
   - open → opening ✓（以元音结尾，不双写）

5. **-ed 的发音规则**：
   - 清辅音后 /t/：walked, helped
   - 浊辅音/元音后 /d/：played, lived
   - /t/ 或 /d/ 音后 /ɪd/：wanted, needed

6. **动词原形不能作谓语**：必须有适当的变化形式才能作谓语。

#### 三、用所给动词的适当形式填空

1. He ____ (work) in this company for 5 years. （现在完成时）
2. I ____ (see) that movie last week. （一般过去时）
3. She ____ (write) a letter now. （现在进行时）
4. The book ____ (write) by a famous author. （被动语态）
5. They ____ (go) to Beijing yesterday. （一般过去时）
6. I have ____ (finish) my homework. （过去分词）

#### 四、选择题

1. She always ____ (watch) TV in the evening.
   A. watch B. watches C. watching D. watched

2. He ____ (go) to school every day.
   A. go B. goes C. going D. went

3. They ____ (play) games when I came in.
   A. play B. plays C. playing D. were playing

4. The work ____ (do) by the students.
   A. is B. are C. was D. were

5. She has ____ (write) three letters.
   A. write B. writes C. written D. writing

#### 五、答案

**一、**
1. works, worked, worked
2. plays, played, played
3. goes, went, gone
4. sees, saw, seen
5. writes, wrote, written
6. makes, made, made
7. takes, took, taken
8. knows, knew, known

**二、**
1. working    2. running    3. making    4. writing    5. swimming    6. studying    7. playing    8. dancing

**三、**
1. has worked（现在完成时，has + 过去分词）
2. saw（一般过去时）
3. is writing（现在进行时）
4. is written（被动语态）
5. went（一般过去时）
6. finished（过去分词）

**四、**
1. B（第三人称单数）
2. B（第三人称单数）
3. D（过去进行时）
4. A（被动语态，单数用 is）
5. C（has + 过去分词）
